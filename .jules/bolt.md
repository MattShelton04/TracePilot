## 2026-05-04 - SQLite dynamically generating large SQL placeholder strings in Rust
**Learning:** When dynamically generating large SQL placeholder strings in Rust for batch inserts, use chunk slice additions via `.push_str()` (e.g., `.push_str(", ?")`) instead of scalar character-by-character pushes (`.push(',')`, `.push('?')`). Also combine repeating prefixes/suffixes (like `, (?,?,?)`) into a single string allocation and append that in inner loops. This avoids per-character checks and significantly boosts execution speed for large string construction.
**Action:** When creating repetitive strings like SQL placeholders in Rust, minimize string mutation operations in loops by grouping `.push` operations into pre-allocated string chunks and applying them via a single `.push_str()`.
