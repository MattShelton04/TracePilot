# Copilot CLI Version Analysis Reports

This folder is reserved for timestamped reports generated by the `tracepilot versions report` CLI command. Historical generated reports were removed during the 2026-05-01 documentation cleanup; generate a fresh report when you need current coverage data.

## Naming Convention

Reports follow the format: `{date}-v{latest_version}-{label}.md`

- **date:** `YYYY-MM-DD` of generation
- **latest_version:** The newest installed Copilot CLI version at time of generation
- **label:** A short descriptor (`baseline`, `post-implementation`, `quarterly-review`, etc.)

## How to Generate a New Report

```bash
# Generate with a timestamped name
pnpm cli versions report --output "docs/reports/versions/$(date +%Y-%m-%d)-v{VERSION}-{label}.md"

# Or on Windows PowerShell:
pnpm cli versions report --output "docs/reports/versions/$(Get-Date -Format yyyy-MM-dd)-v1.0.7-label.md"
```

## When to Generate

- **After adding new event type support** — to document the coverage improvement
- **After a new Copilot CLI version is installed** — to detect schema changes
- **Periodically** — to track the evolving gap as Copilot CLI evolves

## Other Useful Commands

```bash
# Quick coverage check (no full report)
pnpm cli versions coverage

# List installed versions
pnpm cli versions list

# Diff specific versions
pnpm cli versions diff 1.0.5 1.0.7

# Find sessions with a specific event type
pnpm cli versions examples -e session.truncation
```
