Prototype A Slide-out

Refined Stream · conversation default

An evolution of today's slide-out, tightened up. Activity flows chronologically along a vertical rail with status-coloured nodes, restrained type, and micro-animations on row expansion. Prompt and output sit above as collapsible anchors so the timeline never has to compete for visual weight.

Prototype B Inline page

Dossier · agent-tree default

An inline, page-like report with a sticky two-column layout — metadata sidebar (status, identity, lifecycle) on the left, the narrative content (prompt, output, result, reasoning, tools) on the right. Heavy use of cards, KV grids, and status chips. Collapses to a single column under 880px.

Two-column report Sticky meta sidebar Card-per-section Tool list w/ expansion

Prototype C Workspace

Tabbed Workspace · scales to long runs

A more ambitious panel built like a workspace: sticky breadcrumbs (turn → parent agent → this agent), a mini-timeline at the top plotting tool calls across the agent's runtime, and tabs for Overview / Activity / Tools / Reasoning / Raw. Designed for long-running agents with hundreds of tool calls.
