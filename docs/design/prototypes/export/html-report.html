4 · Plan & Todos

Plan

Implement a complete OAuth 2.0 authentication system for the web platform, supporting Google and GitHub as identity providers. The system uses JWT tokens for stateless session management and includes Express middleware for route protection.

The implementation follows the authorization code flow with PKCE for enhanced security. Each provider has its own module implementing a shared interface, making it straightforward to add new providers (e.g., Microsoft, Apple) in the future.

Rate limiting is applied at the route level to prevent brute-force attacks on authentication endpoints. The rate limiter uses a Redis-backed store for consistency across distributed deployments. Integration tests will be written once the core module is stable.
