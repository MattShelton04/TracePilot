Demo Scenarios
🗑️ Delete Worktree
Danger confirmation with a force-delete checkbox option. Used when removing a Git worktree from the session.
📝 Overwrite Template
Warning confirmation for overwriting an existing template. Warns the user the previous version will be lost.
🔄 Reset All Settings
Danger confirmation with an "Are you sure?" prompt. Resets all user preferences and configuration.
Design Notes
Keyboard: Esc closes the dialog. Enter confirms. Tab cycles focus between buttons.
Overlay: Clicking the backdrop overlay dismisses the dialog (equivalent to Cancel).
Accessibility: role="alertdialog", aria-modal="true", aria-labelledby, aria-describedby.
Reduced motion: Animations are disabled when prefers-reduced-motion: reduce is set.
🗑️

Delete Worktree?

This will permanently remove the worktree feat/oauth-login and all uncommitted changes. This action cannot be undone.

📝

Overwrite Template?

A template named "Session Report — Default" already exists. Saving will replace the previous version. The old template cannot be recovered.

🔄

Reset All Settings?

Are you sure? This will restore all preferences, keyboard shortcuts, and display options to their factory defaults. Your session data will not be affected.
