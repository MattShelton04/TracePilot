Toast System — Centered Top Slide

Minimal centered notifications. One toast at a time slides down from the top with a conic-gradient border countdown. New toasts replace the current one.

Positioning
Centered at the top of the viewport with fixed positioning. Pill-shaped (border-radius: 9999px) for a sleek, minimal look.
One at a Time
Only one toast visible. New toasts animate the current one out, then drop in. Creates a calmer notification flow than stacking.
Border Countdown
A conic-gradient border that shrinks as time passes, using a CSS custom property --progress animated via JS requestAnimationFrame.
Accessibility
Container uses role="status" and aria-live="polite". Respects prefers-reduced-motion by disabling animations and the border effect.
