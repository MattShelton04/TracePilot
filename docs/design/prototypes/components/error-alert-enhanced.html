⚠ Enhanced Error Alert — All Variants

Prototype for an enhanced ErrorAlert system. Shows the current minimal implementation for reference, then all proposed variants (inline, dismissible, retryable, banner, compact) across error, warning, and info severities. Each alert supports role="alert" and aria-live for accessibility.
