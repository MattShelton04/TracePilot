Design Notes
Behavior: Clicking a delete/overwrite button toggles the inline confirmation below. Only one confirmation per group can be open at a time.
Keyboard: Escape closes the currently open inline confirmation.
Animation: Slides open via max-height + opacity transition. Respects prefers-reduced-motion.
Accessibility: role="alertdialog" on the confirm panel, aria-expanded on triggers, aria-controls linking trigger to panel.
Use case: Best for list items, cards, and settings rows where a full modal would be disruptive.
