Toast System — Stacked Bottom-Right

Classic stacked notification pattern. Toasts slide in from the right, stack from the bottom, and auto-dismiss with a progress indicator.

Positioning
Bottom-right fixed position with column-reverse stacking. New toasts appear at the bottom, pushing older ones upward. Max 4 visible.
Animation
Slide-in from right on enter using cubic-bezier easing. Fade-out + slight rightward shift on dismiss. Respects prefers-reduced-motion.
Auto-Dismiss
Progress bar shrinks from full width to zero over the configured duration. Pauses on hover. Resumes when cursor leaves.
Accessibility
Toast container uses role="log" and aria-live="polite". Each toast has a dismiss button with aria-label.
