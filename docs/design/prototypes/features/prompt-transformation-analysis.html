Recommendations
💡
Reduce session reminders. Reminders account for 22-25% of context by turn 10. Consider pruning stale reminders to save ~2,400 tokens per turn.
📦
Compress environment context. Environment data adds ~2K tokens per turn. Summarizing file listings and reducing path verbosity could cut this by 40%.
⚠️
Attachment growth is unbounded. Attachments grow from 5% to 23% of context. Consider a cap at 3 attachments or auto-summarization.
🔄
Use SQL state snapshots. SQL state grows monotonically. Replacing full state with diffs after turn 5 would save ~1,200 tokens per turn.
📊
Consider compaction at turn 8. The expansion ratio exceeds 50× after turn 8. Triggering auto-compaction here aligns with the success rate drop-off.
